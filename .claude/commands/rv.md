`resume-reviewer-swe` 에이전트를 사용해 이력서를 종합 리뷰하세요.

**입력:** $ARGUMENTS

## 처리 방법

1. `$ARGUMENTS`가 비어있다면 사용자에게 이력서 본문을 붙여넣거나 파일 경로를 알려달라고 요청하세요.
2. `$ARGUMENTS`가 파일 경로처럼 보이면 (`/`, `.pdf`, `.md`, `.txt` 포함) Read 도구로 먼저 내용을 읽으세요.
3. `resume-reviewer-swe` 에이전트를 Agent 도구로 실행하세요. 프롬프트에 반드시 포함할 것:
   - **전체 이력서 원문** (Read로 읽었거나 사용자가 붙여넣은 그대로)
   - **타겟 역할** (있다면 사용자가 명시한 그대로; 없다면 에이전트가 후보자에게 묻도록 지시)
   - 에이전트의 10단계 프레임워크를 **모두** 수행할 것 — 특히 ATS 분석, Bullet Rewrite, Hiring Manager Simulation, Action Plan
   - **이력서가 한국어면 피드백도 한국어로**, 영어면 영어로 작성할 것
4. 에이전트 결과가 돌아오면 그대로 사용자에게 전달하되, 가장 중요한 3가지 액션 아이템을 맨 위에 요약해 주세요.
