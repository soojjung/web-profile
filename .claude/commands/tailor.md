`resume-tailoring-strategist` 에이전트를 사용해 마스터 이력서를 특정 채용 공고에 맞춰 재구성하세요.

**입력:** $ARGUMENTS

## 처리 방법

1. **마스터 이력서**와 **채용 공고(JD)** 두 가지를 모두 확보:
   - 파일 경로 → Read
   - URL → WebFetch
   - 본문 → 그대로
   - 둘 중 하나가 없으면 명확히 요청
2. `resume-tailoring-strategist` 에이전트를 Agent 도구로 실행. 프롬프트에 반드시 포함:
   - **마스터 이력서 전체**
   - **JD 전체** (회사명/직무명 포함)
   - 8단계 방법론을 **모두** 수행할 것
   - **절대 사실을 날조/과장/추론하지 말 것** — 누락된 정보는 "gap"으로 표기
   - 허용되는 작업: 순서 재배치, 표현 재작성, 관련성 강조, 잠재 성과 발굴, 진실에 부합하는 ATS 키워드 정렬
   - 언어: JD가 한국어면 한국어로, 영어면 영어로
3. 결과를 받으면:
   - **변경 요약** (어떤 bullet을 어떻게 바꿨고 왜 바꿨는지) 먼저 출력
   - **재구성된 이력서 전문**을 그 아래에 출력
   - 사용자가 원하면 `~/Desktop/개인플젝/mypage/`에 `resume-tailored-{회사명}.md` 형태로 저장 가능하다고 안내 (요청 시 Write)
